## Слайд 3 — Боль (1 мин) ⭐ ключевой

«Начнём с боли, которую описал заказчик в брифе.

**Служба поддержки тонет в однотипных вопросах.** Ответы уже есть — в Functionalnie.docx, в регламенте командировок, на странице новичка. Но пользователь не лезет в PDF: он пишет оператору.

Четыре проблемы:

1. **Документация огромная** — четыре файла кейса, сотни страниц.
2. **Люди не ищут сами** — „Как работает фильтр?“ „Кто согласует командировку?“
3. **Ночью и в пик** — вопрос висит до утра.
4. **Оператор = копипаст** — одни и те же ссылки из документов.

**Нам нужна первая линия**, которая отвечает за секунды и знает, когда передать человеку.»

## Слайд 5 — Решение (1 мин) ⭐ ключевой

«**DocHelper** — Telegram- и web-бот, первая линия с RAG.

1. Пользователь пишет **как оператору** — текст, голос, скриншот.
2. Бот **понимает контекст** — неформально, с опечатками.
3. **За секунды** находит фрагмент и даёт **ответ с цитатой, источником и скачиванием docx**.
4. Низкая уверенность → **оператор**, история сохранена.»

## Слайд 6 — Архитектура · закрытый контур (45 сек)

Для **продакшена** — Ollama, Whisper, Redis локально. Для **demo сегодня** — GigaChat, переключение одной строкой в `.env`. Это прямое требование брифа про MML AI без интернета.»

## Слайд 7 — Live demo (2 мин) ⭐ главное время

«Показываем **живой продукт**, не скриншоты. Жюри может повторить с QR.

1. **Telegram** — `/start`, вопрос *«Как работает фильтр документа?»* → ответ + **Functionalnie.docx** + фрагмент.  
   *Зачем:* RAG работает, источник прозрачен.

2. **Web-чат** — тот же вопрос → блок с фрагментом из документации.  
   *Зачем:* второй канал для пользователей БАРС-Офис.

3. **Эскалация** — кнопка «Оператор» → панель `/operator`, дежурный отвечает.  
   *Зачем:* бот не врёт при низкой confidence, оператор видит историю.

4. **Аналитика** — 94%, топ вопросов.  
   *Зачем:* заказчик видит, что автоматизируется.

*Если успеем — голос про командировку или скриншот. Если нет — QR на первом слайде.*»

## Шпаргалка: ответы на типичные вопросы жюри

**«Это не ChatGPT?»**  
→ RAG-only: system prompt запрещает ответ вне контекста. Confidence gate. Обязательный источник.

**«Откуда 94%?»**  
→ 17 тест-кейсов из брифа, скрипт `scripts/test_brief_cases.py`, 16 авто + 1 эскалация.

**«Без интернета?»**  
→ Ollama + e5 + Qdrant + Whisper на сервере заказчика.

**«А если документ обновился?»**  
→ Переиндексация через админку; в roadmap — webhook из Git.

**«Галлюцинации?»**  
→ Порог confidence 0.45–0.55; ниже — оператор.

Наше решение

DocHelper — первая линия с RAG

  1. Пользователь пишет как живому оператору — текст, голос, скриншот
  2. Бот понимает контекст, опечатки, неформальные формулировки
  3. Ищет в docx/pdf · ответ + цитата + источник + скачать
  4. Низкая уверенность → кнопка «Оператор» · история сохранена

## Слайд 2 — Проблема

**Служба поддержки тонет в однотипных вопросах**

- Документация объёмная и постоянно обновляется
- Пользователи не ищут сами — пишут оператору
- Вне рабочего времени вопросы «висят» до утра
- В пик (релизы, акции) команда не справляется
- Операторы копируют одни и те же ссылки из PDF

**→ Нужна первая линия, которая отвечает мгновенно и не выдумывает**

---

## Слайд 3 — Решение

**DocHelper Барс — умный помощник в Telegram и Web**

Пользователь пишет как живому оператору. Бот:

- понимает вопрос на естественном языке (и **голосом**);
- ищет ответ в **актуальной документации** (RAG);
- даёт ответ с **источником** (название docx);
- при низкой уверенности — **переключает на человека** с историей диалога.

**Цель брифа:** ≥ 40% автоматизации первой линии  
**Наш результат на тест-кейсах:** **94%** (16/17)

## Слайд 4 — Архитектура

```
┌─────────────┐     ┌──────────────┐     ┌─────────┐
│  Telegram   │────▶│   FastAPI    │────▶│ Qdrant  │
│  + Web UI   │     │   (RAG)      │     │ (e5)    │
└─────────────┘     └──────┬───────┘     └─────────┘
                           │
              ┌────────────┼────────────┐
              ▼            ▼            ▼
         GigaChat      Ollama       Whisper
         (демо)    (закрытый контур)  (голос)
              │            │            │
              └────────────┴────────────┘
                           ▼
                    Ответ + Источник
                           │
              confidence < порог?
                           ▼
                    Эскалация → Support
```

**Стек:** Python · FastAPI · Qdrant · Redis · Docker · GigaChat · Ollama · Whisper

## Слайд 7 — AI-агенты

| Агент | Задача |
|-------|--------|
| **Speech** | Whisper: голос → текст |
| **Retriever** | e5 + Qdrant: поиск фрагментов |
| **Generator** | GigaChat/Ollama: ответ по контексту |
| **Evaluator** | Confidence: авто или оператор |
| **Escalation** | История → support-чат |
| **Analytics** | Метрики, слабые места документации |

**Не chatbot — orchestrated AI pipeline**

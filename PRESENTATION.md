## Слайд 6 — Live demo

Слева — сценарий, справа — **видео** (если файл есть).

**Как добавить видео:**
1. Запишите экран (1–2 мин): фильтр → web → эскалация → оператор.
2. Сохраните как `static/presentation/demo.mp4` (или `demo.webm`).
3. Опционально: обложка `static/presentation/demo-poster.jpg`.
4. Откройте http://127.0.0.1:8026/presentation — плеер появится сам.

На защите: **live по QR** или **Play** на записи, если сеть подведёт.
